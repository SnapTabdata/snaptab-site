# Gitee Pages 开启步骤（官网国内访问）

代码已推送到 Gitee，按以下步骤开启 Pages 后，官网即可在国内直接访问（无需梯子）。

## 操作步骤

1. 打开仓库：https://gitee.com/liujing5573/snaptab-site
2. 点击左侧菜单 **「服务」** → **「Gitee Pages」**
3. 选择分支：**master**
4. 选择部署目录：**/（根目录）**
5. 点击 **「启动」** 或 **「更新」** 部署

## 部署完成后

- **官网地址**：https://liujing5573.gitee.io/snaptab-site/
- 国内用户可直接访问，无需梯子
- 下载页已支持 GitHub + Gitee 双源

## 后续更新

每次 `git push gitee master` 后，Gitee Pages 会自动重新部署（或需在 Pages 页面手动点「更新」）。
