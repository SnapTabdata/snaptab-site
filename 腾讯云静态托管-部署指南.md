# SnapTab 官网部署到腾讯云静态托管

腾讯云静态托管（云开发）提供免费默认域名，国内访问稳定，无需梯子。

## 一、前置准备

1. **腾讯云账号**：https://cloud.tencent.com/
2. **实名认证**：需完成实名认证才能开通
3. **免费额度**：新用户 1GB 容量 + 5GB 流量/月，约 0.21 元/天起

## 二、控制台部署（推荐新手）

### 步骤 1：创建云开发环境

1. 打开 [云开发控制台](https://console.cloud.tencent.com/tcb/env/index)
2. 点击「新建环境」
3. 选择「按量计费」，地域选「上海」
4. 环境名称填 `snaptab-site`，创建

### 步骤 2：开通静态网站托管

1. 进入环境后，左侧菜单点「静态网站托管」
2. 点击「开通」
3. 开通后进入托管管理页

### 步骤 3：上传网站文件

1. 点击「上传文件夹」
2. 选择本地的 `snaptab-site` 文件夹（包含 index.html、download.html、assets 等）
3. 上传完成后，点击「设置」→ 索引文档填 `index.html`

### 步骤 4：访问网站

- 默认域名格式：`环境ID.tcloudbaseapp.com`
- 在托管页可复制访问地址

## 三、CLI 一键部署（推荐开发者）

```bash
# 1. 安装 CloudBase CLI
npm install -g @cloudbase/cli

# 2. 登录（会打开浏览器授权）
cloudbase login

# 3. 进入网站目录并部署
cd d:\Market_Research_Auto\snaptab-site
cloudbase hosting deploy . -e 你的环境ID
```

环境 ID 在云开发控制台 → 环境设置 → 环境 ID 中查看。

## 四、自定义域名（可选）

若有自己的域名（如 snaptab.cn）：
1. 静态托管 → 域名管理 → 添加域名
2. 按提示在域名服务商处添加 CNAME 解析
3. 可申请免费 SSL 证书，开启 HTTPS

## 五、后续更新

- **控制台**：重新上传覆盖即可
- **CLI**：再次执行 `cloudbase hosting deploy . -e 环境ID`
